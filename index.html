OpenTimestamps.org is a trustless timestamping protocol using Bitcoin as a Notary

Use this tool to prove or verify a proof that a given file existed at the time it was timestamped. Opentimestamps servers aggregate an unlimited number of file hashes into a merkle tree. The merkle root is committed to the Bitcoin blockchain via a Bitcoin transaction, and the server returns an OTS file. The OTS file is a log of the operations used by the server in the timestamping process. The OTS client (used in this online tool) verifies the proof-of-existence of your file using the OTS file as reference. The OTS client connects to a full node to ensure that the merkle root is actually committed to the blockchain, and checks that your file hash was actually included in that merkle tree.
